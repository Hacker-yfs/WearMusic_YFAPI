# WearMusic

## 🏠简介

一款网易云超强第三方客户端，拥有歌词、进度调整、查看评论等官方客户端没有的功能！



## 💻参数

屏幕支持：方屏/圆屏（自适应）
音乐来源：网易云音乐



## ❔常见问题

##### 为什么选择WearMusic？

UI美化度高、功能多、更新频繁、具有更方便的反馈通道。

##### 如何使用？

Star本仓库，前往Release下载。
