## 应用详情

### 应用简介

一款第三方网易云音乐客户端，支持歌词、音乐云盘、歌单、评论等功能。

已修改为Hacker_yfs自建API
原作者JackuXL保留此项目所有权

项目原地址：[GitHub](https://github.com/OpenSourceWear/WearMusic)
